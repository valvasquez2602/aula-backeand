## Step By Step Config Express Server

### Step 1: Estar dentro da pasta e/ou repositório
```
Caso não esteja dentro da pasta ou repositorio , encontre-o e acesse-o
```

### Step 2: Instalar dependências
```
npm i express
```

### Step 3: Organizar as pastas e arquivos base do projeto

```
Criar a pasta src... routes , server.js e etc
```

### Step 4: Configurar o package.json

```
deixar o type: "module" e configurar o(s) scripts ex: (node --watch ./src/server.js)
```

### Step 5: Configurar o arquivo server.js

```
Configurar o servidor, ou seja, importar o express , configurar o middleware para ler o json, definir e executar a escuta da porta
```

### Step 6: Continuar com o projeto

```
Criar os arquivos dentro das pasta routes e services, importar a rota no server e repetir o mesmo proscesso para todos os demais módulos
```
